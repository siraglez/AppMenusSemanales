# AppMenusSemanales 🍎🥗

> **Trabajo de Fin de Grado (TFG)**
> Aplicación nativa para iOS desarrollada con SwiftUI y SwiftData para la planificación automática de menús semanales.

## 📋 Descripción del Proyecto

Este proyecto consiste en el desarrollo de una aplicación móvil que permite generar menús semanales personalizados, adaptados a las preferencias del usuario, restricciones dietéticas y estaciones del año.

El objetivo principal es resolver la problemática de la planificación alimentaria mediante un algoritmo que crea dietas equilibradas y gestiona recetas de forma eficiente.

## 🚀 Funcionalidades Principales

El alcance del proyecto incluye las siguientes características clave:

* **Gestión de Recetas:** Añadir, modificar y eliminar recetas indicando tipo de comida, estación e ingredientes.
* **Generador Automático de Menús:** Algoritmo inteligente que crea una planificación semanal equilibrada respetando reglas específicas.
* **Persistencia de Datos:** Almacenamiento local seguro de recetas y menús utilizando **SwiftData**.
* **Interfaz Intuitiva:** Diseño limpio y amigable desarrollado con **SwiftUI**.

## 🛠 Stack Tecnológico

* **Lenguaje:** Swift
* **Framework UI:** SwiftUI
* **Persistencia:** SwiftData
* **Arquitectura:** MVVM (Model-View-ViewModel)
* **Plataforma:** iOS

## 📂 Estructura del Proyecto

El código fuente se organiza siguiendo el patrón MVVM:

* **Models:** Definición de datos (Recetas, Menús).
* **Views:** Interfaz de usuario (Pantallas y Componentes).
* **ViewModels:** Lógica de negocio y Algoritmo de generación.
